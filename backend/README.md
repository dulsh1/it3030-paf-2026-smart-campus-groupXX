# Backend (Spring Boot REST API)

## Module Overview
This folder contains the Spring Boot REST API for Smart Campus Operations Hub.

## Recommended Packages (example)
- `controller`
- `service`
- `repository`
- `model` / `entity`
- `dto`
- `config` (security, CORS)

## Notes
- Configure DB connection in `src/main/resources/application.properties`
- Use MySQL 8+
